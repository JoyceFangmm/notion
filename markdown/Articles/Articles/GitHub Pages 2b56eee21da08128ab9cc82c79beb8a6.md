# 仓库规则

分为两种：

1. 直接使用用户名来命名
    1. `<用户名>.github.io` 仓库
    2. 访问地址`https://<用户名>.github.io`
2. 普通仓库
    1. 比如Notion
    2. 访问地址`https://<用户名>.github.io/<仓库名>` 
    3. 注意大小写，如果仓库名包含大写，地址也需要是大写
    4. [https://joycefangmm.github.io/notion/](https://joycefangmm.github.io/notion/)
    5. [https://joycefangmm.github.io/tools/](https://joycefangmm.github.io/tools/)
