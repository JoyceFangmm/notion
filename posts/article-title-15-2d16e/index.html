常用&主流LLM深入了解

  • Claude 4.6 (Anthropic)

    • 旗舰版 (Claude Opus 4.6)，性价比版 (Claude Sonnet 4.6)
    • Cursor内置并深度集成了Cluade模型。Cursor是IDE
    • Claude Code：底层：Claude Opus/Sonnet/Haiku 代码专项微调；形态：CLI 终端 + VS Code/JetBrains IDE 插件 + 桌面 / 网页端；

  • GPT-5 (OpenAI)

    • GPT-5.5 Instant，ChatGPT 默认模型
    • GPT-5-Codex-Mini、GPT-5.2-Codex、GPT-5.3-Codex等全都融入进了GPT-5.4、GPT-5.5内了
    • OpenAI Codex，三种主要方式：
      • Codex CLI（命令行工具），直接在你的终端里运行 npm install -g @openai/codex 安装。它能读取、编辑代码文件，并从自然语言指令中执行命令
      • Codex IDE (集成开发环境插件)，在VS Code等主流编辑器中直接使用
      • Codex App (独立桌面应用)，这是Codex作为通用助手的完整形态，提供了GUI界面，方便操控电脑

  • 智谱AI (GLM系列)

    • GLM-5v-Turbo<多模态，图片输入，推理模型>

    • GLM-5.1

    • GLM-5.0-Turbo

    • GLM-5.0

  • DeepSeek (深度求索)

    • DeepSeek-V4<DeepSeek-V4-pro、DeepSeek-V4-flash>
    • DeepSeek-R1
    • DeepSeek-V3.2

Graphify

推荐一个超级省钱的skill，在日常编码中，或者项目review中，可以节省大量的token：
https://github.com/safishamsi/graphify/blob/v7/docs/translations/README.zh-CN.md

Graphify 是一款面向 AI 编程助手的 Skill 插件，由伦敦 Valent 公司 AI 研究员 Safi Shamsi 开发。它通过 /graphify 命令将任意文件夹（代码、文档、论文、图像、音视频）转换为可查询的知识图谱，帮助 AI 助手更高效地理解和处理复杂项目。
